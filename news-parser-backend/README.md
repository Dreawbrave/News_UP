# news-parser-backend

Простой backend для парсинга текста новостных статей по URL.

## Установка зависимостей

```
npm install
```

## Запуск

```
npm start
```

## Использование

GET-запрос:
```
http://localhost:3001/parse?url=ССЫЛКА_НА_НОВОСТЬ
```

Ответ:
```
{
  "text": "...текст статьи..."
}
```

> Для разных сайтов может потребоваться доработка парсинга (селекторы).
